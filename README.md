# E-Commerce SQL Project

This project is designed to demonstrate SQL skills through analysis of a simulated e-commerce database system. It uses realistic Indian customer and product data, structured across multiple CSV files.

## 📁 Dataset Files

- `Customers.csv` — Customer personal and contact information  
- `Products.csv` — Product listings with categories and prices  
- `Orders.csv` — Purchase orders placed by customers  
- `Order_Items.csv` — Items within each order (quantity, product)  
- `Product_Reviews.csv` — Customer reviews for products  

## 📊 Use Cases

This project can help you practice:
- SQL Joins
- Aggregation and Grouping
- Subqueries and Nested queries
- Window functions
- CTEs (Common Table Expressions)
- Data filtering and manipulation

## 📌 Sample Schema

- ER diagram is available in `ecommerce_er_diagram.png`
- Includes relationships between Customers, Products, Orders, Reviews, and Order Items

## ⚙️ Tools

- MySQL or any SQL-compatible engine
- MySQL Workbench (recommended)
- GitHub for version control
